Driving School API

Ruby on Rails 8.1 · PostgreSQL · Puma · Active Storage

Base URL: http://localhost:3001 CORS: http://localhost:3000 Format: JSON

Health

GET /up Returns 200 OK if the app is live

Authentication

POST /api/v1/auth/login Log in with email/password, returns JWT
POST /api/v1/auth/register Self-register as student role, returns JWT
DELETE /api/v1/auth/logout Log out (token added to denylist)
GET /api/v1/auth/me Get current authenticated user's profile

Auth-required endpoints need an Authorization: Bearer <token> header. All auth endpoints return a standardized envelope:

{
  "success": true,
  "data": { ... },
  "message": "Login successful"
}

exam_booking[notes]textnoAdditional notes

Eligibility Requirements

The student must pass all of these checks before a booking is created:

  • Status must be exam_eligible
  • theory_days_completed >= 35
  • practical_days_completed >= 52
  • mock_test_score > 37

Response — 403 Forbidden

{
  "errors": [
    "Student status 'registered' is not eligible for exam booking",
    "Theory training incomplete: 10/35 days required",
    "Practical training incomplete: 5/52 days required",
    "Mock test score insufficient: 20% (minimum 38% required)"
  ]
}

Cancel Booking

POST /api/v1/students/:student_id/exam_bookings/:id/cancel

Sets status to cancelled. No request body required.

Record Result

POST /api/v1/students/:student_id/exam_bookings/:id/record_result

Parameters

FieldTypeReqDescription
exam_booking[score]integeryes0-100
exam_booking[notes]textnoResult notes

Sets status to completed and records the score. Passing threshold: >= 50. If the score is below 50, a 7-day penalty is applied to the student.

Penalty System

When a student fails an exam (score < 50), a 7-day penalty is automatically applied, blocking further exam bookings during that period.

{
  "under_penalty": true,
  "penalty_start_date": "2026-06-25T10:00:00Z",
  "penalty_end_date": "2026-07-02T10:00:00Z",
  "penalty_reason": "Failed theory exam on 2026-06-25"
}

Penalty is considered active when under_penalty == true and penalty_end_date is in the future.
